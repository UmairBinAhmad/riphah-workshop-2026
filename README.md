# Riphah Workshop 2026

This repository contains workshop materials and notebook-based examples focused on image classification with Convolutional Neural Networks (CNNs). CNNs are deep learning models that automatically learn important visual features such as edges, patterns, textures, and shapes from images. Because of this, they are widely used for solving computer vision problems like image recognition and classification.

Using practical examples inspired by introductory CNN resources and TensorFlow image classification tutorials, this workshop is designed to help learners understand how image classifiers are built, trained, and evaluated. The code in this repository provides a hands-on starting point for exploring deep learning concepts and applying them to real-world image datasets.
